# テストデータについて
src/modules/testdata/generate_test_data.py により作成したテストデータを使用します。
ファイルは src/modules/testdata/dataset1.tsv, dataset2.tsv, dataset3.tsv の3つです。
テストデータは以下の構成です
- 1.ソート対象配列
- 2.ソート後の配列
- 3.ソートインデックス

ソートした後に「2.ソート後の配列」と比較して作成したロジックが正しい事を確認します。

# 出力について

実行出力の内容は以下の通りです。
size = データセットのサイズ
distance = ソート前とソート後の距離（0.0が完全に同一）、ソートミスがあった場合には距離が大きくなります。
sorted top 5 = ソート後の上位5件
sorted tail 5 = ソート後の下位5件
elapsed = ソートにかかった時間（秒）

# distance についての注意
numpy で float64 → float32 の型変換や、JSON等他形式へ変換後に取り込み直すみたいな処理がはさまるとソートが正しくても distance が 0.0 にならない場合があります。
コード修正する際は数値形式について留意すること。
今回作成したコードは素直に np.save -> np.load しているため、distance は 0.0 になります。
